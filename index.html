Munki

Welcome to Munki!

Munki is the most widely used open source software deployment tool for the OS X platform. Munki is surrounded by a vibrant community that is passionate about deploying software on OS X while providing an excellent user experience.

Getting started

To get started with Munki, jump over to the Munki Wiki. There is thorough documentation on how to use every facet of the Munki project.

Community

Part of what makes Munki so popular is the fantastic community that surrounds it. If you want to get involved in the community, you can join the mailing list, pop into the ##osx-server IRC channel on Freenode or come to one of many Mac Admin conferences around the world.
